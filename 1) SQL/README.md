# SQL


## Задание
Есть таблица (`./data/Таблица_SQL.csv`) с историей операции по кредитам со следующими полями: 
* CRED_ID — id
* DATA — дата операции
* OPER — тип операции:
    * Вынос на просрочку — просрочка увеличивается в определенную дату на определенную сумму
    * Гашение просрочки — просрочка уменьшается в определенную дату на определенную сумму
* SUMMA — сумма операции

**Нужно из этой таблицы получить диапазоны просрочки для каждого кредита**, в данном случае, такую таблицу:

```
CRED_ID	| DATA_BEGIN |  DATA_END
――――――――┼――――――――――――┼――――――――――――
   1	| 2020-01-01 | 2020-01-11
   1	| 2020-01-16 | 2020-02-15
   2	| 2020-01-05 |    NULL
   3	| 2020-01-01 |    NULL
```

Где:
* DATA_BEGIN — дата, когда осуществляется первый вынос на просрочку или первый вынос после полного гашения просрочки 
* DATA_END — дата, когда просрочка полностью выгашивается


**Пояснение**:
* По кредиту 1 видно, что клиент выходил и гасил просрочку в ноль два раза.
* По 2 и 3 кредиту не было полного гашения, а значит просрочка активна на текущий день (дата закрытия NULL).
